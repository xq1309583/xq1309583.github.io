Microsoft Edge浏览器收藏夹同步

发表于 2019-08-15 更新于 2019-08-30 分类于 技术

我一直用的谷歌浏览器，电脑的收藏夹咔一下就可以同步，但是手机的咔一下不行，还要谷歌全家桶才能开启同步功能。安卓的Edge浏览器虽然早就用上了谷歌内核，但是经常卡死（Miui Android9）。现在电脑的Edge也要用上谷歌内核了，手机浏览器八成也有救了。

都装上先同步一下收藏夹试试，发现同步的是一些年代久远的网站……虽然我用Edge导入了一大堆，网上说要等几天，结果还是不行。偶然有一天上某网站打开了IE，发现手机Edge同步的收藏夹正是IE里面的。原来电脑的Edge并不会和IE互相同步收藏夹，手机也会优先同步IE的收藏夹。那我每次更新了收藏夹还要上IE再导入？

解决方法

把IE的收藏夹清空，对应的收藏夹那个文件夹也清空，Edge就可以正常同步了。

速度挺快的，也就几分钟，没有像网上说的那样几天才有。
